# SubVert
As in **Sub**title con**Vert**er

> [!WARNING]
> Unstable software. Heavy work in progress. Use at own risk.

## Build
```
./build.sh
```

## Usage
```
subvert [-o output ] input.{vtt,srt}
```

## Supported format conversion (Not fully compatible yet)
- vtt to lrc
- vtt to srt
- srt to vtt
